# i18n 진행 현황

`next-intl` (웹) + `i18next` (모바일) 기반 다국어 준비 작업.  
지원 언어: **영어(en, 기본값)**, 한국어(ko)

---

## 완료된 작업

### 웹 인프라
- `apps/web/i18n/routing.ts` — `locales: ['en', 'ko']`, `defaultLocale: 'en'`
- `apps/web/i18n/request.ts` — `getRequestConfig` 설정 (메시지 파일 동적 import)
- `apps/web/middleware.ts` — `createIntlMiddleware` 통합, `/checkin` → `/en/checkin` 리다이렉트, Supabase 세션 처리 공존
- `apps/web/app/[locale]/` 구조로 전체 페이지 이동 완료

### 메시지 파일
- `apps/web/messages/en.json`, `ko.json` (104줄) — common, category, home, trip, checkin, location, settings, auth, story, calendar 네임스페이스
- `packages/shared/messages/en.json`, `ko.json` — 카테고리 레이블 (웹·모바일 공통)

### 모바일 인프라
- `apps/mobile/src/i18n/index.ts` — `i18next` + `expo-localization` 초기화, 디바이스 언어 자동 감지
- `apps/mobile/src/i18n/en.json`, `ko.json` (76줄) — common, category, tab, home, trip, checkin, schedule 네임스페이스
- `apps/mobile/App.tsx` — `I18nextProvider` 루트 주입 완료

### 웹 컴포넌트 — `useTranslations()` 적용 완료 (5개)
- `app/[locale]/checkin/page.tsx`
- `app/[locale]/checkin/components/TripContent.tsx`
- `app/[locale]/checkin/components/EmptyTripsView.tsx`
- `app/[locale]/checkin/components/CheckinTimeline.tsx`
- `app/[locale]/login/page.tsx`
